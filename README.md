## Описание проекта

Apollo 11 Archive представляет собой веб-приложение, посвящённое истории космической миссии Apollo 11 (июль 1969 года). Проект был создан в образовательных целях для популяризации научных достижений в космонавтике и сохранения исторической информации о первых людях, вступивших на поверхность Луны.

### Основные возможности

- Подробная хронология событий миссии Apollo 11
- Информация о членах экипажа (Нил Армстронг, Базз Олдрин, Майкл Коллинз)
- Технические характеристики ракеты Saturn V и космических аппаратов
- Галерея исторических фотографий и архивных материалов
- Журнал логов и записей переговоров миссии
- Современный дизайн с поддержкой тёмной темы
- Полная адаптивность для мобильных и стационарных устройств

## Описание страниц

| index.html | Главная страница с хронологией событий миссии Apollo 11 |

## Технологический стек

- HTML5 (семантическая структура документов)
- CSS3 (современное оформление с поддержкой переменных и медиа-запросов)
- Firebase Hosting (облачный хостинг и развёртывание)
- Responsive Web Design (адаптивный дизайн для всех устройств)


## Руководство пользователя

Основные сценарии использования:

1. **Ознакомление с хронологией**: На главной странице представлена полная временная шкала событий миссии с описанием ключевых этапов

2. **Изучение информации об экипаже**: На странице crew.html находятся подробные сведения о каждом членом команды

3. **Просмотр технических данных**: Раздел tech.html содержит подробные технические характеристики оборудования

4. **Изучение архивных материалов**: Галерея и журнал логов предоставляют доступ к исторической информации и документам

## Характеристики оформления

Сайт поддерживает темный вариант оформления:

- Тёмное оформление (по умолчанию): тёмный фон с акцентными цветами (голубой и оранжевый)
- Оптимальное отображение на всех размерах экрана (десктоп, планшет, смартфон)
